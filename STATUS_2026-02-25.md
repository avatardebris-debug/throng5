## 1) What This Project Is

**Throng4** — a human-imitation RL framework for Atari games, focused on Montezuma's Revenge.
The goal is to train an agent that can complete Room 0 (collect key → exit door) using:
- Behavioral Cloning (BC) from human play → RL fine-tuning with shaped rewards
- A World Model (DreamerEngine) for compressed simulation / MCTS guidance
- Subgoal detection and novelty tracking for sparse-reward navigation
- Meta-learning (MetaAdapter + PrincipleStore) to transfer policies across games

### Pipeline

```
Human plays → replay_db.sqlite → BC pretrain
→ montezuma_explorer.py --use-dreamer
  → WorldModel calibration → Dyna-Q synthetic rollouts
  → MCTS prior with action boost
  → Shaped rewards (subgoals + novelty + directional bias + fall penalty)
  → Semantic logs → generate_crossgame_brief → Tetra → PRINCIPLE/ADD ops
  → ingest_crossgame_inbox → PrincipleStore + atari_hypotheses
  → MetaAdapter reads PrincipleStore → next training run
```

|---|---|---|

## 8) Key Design Decisions & Rationale

| Decision | Rationale |
|---|---|
| Stage-aware directional bias (not just key-aware) | Pre-key, agent needs to go RIGHT first (to rope), not left. Left bias was counterproductive. |
| MCTS action boost > reward shaping alone | Q-values haven't learned jump-right yet. Prior boost forces MCTS to explore that path. |
| Synthetic transitions at 1/11 weight | Real>synthetic. Hallucinations shouldn't dominate training. |
| Confirmation via cosine similarity ≥0.85 | Perfect match unlikely; directional match sufficient to confirm WorldModel prediction. |
| Fall penalty without simulating to death | `is_falling(prev_y, y)`: dy>20 ≈ falling. Safe band x=67–145 (to/from rope). Avoids needing full rollout to detect lethal state. |
| MCTS arc-exception for void zone | Void zone is lethal only on unsafe falls, not on mid-arc jumps. Top-platform long arc (dy=30+, x=77→109) is explicitly allowed via safe x-band. |
| Coordinates from RAM not pixels | RAM bytes 42/43 give exact agent position; no need for CNN pixel parsing for basic navigation. |
| Checkpoint always loads (no rejection) | Saved states via F5 are always valid. Engine handles mid-air restores gracefully. |
| Novelty bin 5px (was 8px) | Finer-grained exploration credit encourages more continuous movement. |
| Movement bonus +0.03/step | Distinct from stall penalty (fires after 300+ idle steps). Positive reinforcement for any motion. |
