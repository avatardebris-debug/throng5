# Project Status — 2026-02-22 (updated 20:48)

## Where We Are

Throng4 is a human-imitation RL framework for Atari games with a meta-learning
layer (DreamerEngine + OptionsLibrary) that self-adapts to environment type.

### Pipeline

```
Human plays → replay_db.sqlite → BC pretrain → montezuma_explorer --use-dreamer
→ Semantic logs → generate_crossgame_brief → Tetra → PRINCIPLE/ADD ops
→ ingest_crossgame_inbox → PrincipleStore + atari_hypotheses
→ MetaAdapter reads PrincipleStore → next training run
```

### Bug fixed: dreamer_reliance stuck at 0.5
**Root cause:** `DreamerTeacher._agreement_history` seeded with `[True, False]` neutral
prior, but `record_policy_action()` was never called in `round_robin_runner.py`.
**Fix:** Added `dreamer_teacher.record_policy_action()` call after each step.
Reliance will now correctly evolve away from 0.5 as the agent learns.
