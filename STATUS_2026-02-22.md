# Project Status — 2026-02-22

## Where We Are

Throng4 is a human-imitation RL framework for Atari games. The pipeline runs:

```
Human plays → replay_db.sqlite → Imitation training → Agent evaluates
→ Semantic logs (with decoded RAM state) → Tetra brief → Tetra hypotheses
→ Next training run
```

## Architecture Notes

The agent is `PortableNNAgent` (256→128→1 network, RAM+one-hot action input).
- `forward(x)` — single scalar Q-value
- `forward_batch(X)` — batch, numerically identical, same speed at this network size
- `save_weights(path)` / `load_weights(path)` — .npz format

The imitation head (`Wi1/Wi2`) is a separate classification head that outputs
action probabilities. It trains via cross-entropy. The backbone Q-values train
via Bellman updates. BC pretraining trains the backbone directly.
